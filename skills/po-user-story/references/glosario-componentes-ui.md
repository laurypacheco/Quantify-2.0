# Glosario de Componentes UI

Fuente: GUÍA-QA-Términos y conceptos de sistemas v1.00, §2-4. Usar estos términos (nombre en
inglés + traducción al español entre paréntesis) al redactar criterios de aceptación y
resultados esperados, para que QA y DEV interpreten igual los nombres de los componentes.

> Idioma de nombres de componentes: los nombres de los componentes utilizados siempre deben ir en
> el mismo idioma, para mantener consistencia dentro de una misma historia/criterio.

## Partes de las páginas

- **Header** (encabezado)
- **Navigation Bar** (barra de navegación)
- **CTA / Call to action** (llamada a la acción)
- **Body** (cuerpo)
- **Sidebar** (barra lateral)
- **Footer** (pie de página)

## Tipos de páginas

- **Login** (inicio de sesión)
- **Home** (inicio / principal)
- **Dashboard** (cuadro de mando)
- **Search** (búsqueda)
- **Forms** (formularios)
- **Reports** (informes)

## Campos

- **Text field** (campo de texto)
- **Dropdown** (desplegable / listado)
- **Checkbox** (casilla de verificación)
- **Date picker** (selector de fecha / calendario)
- **File upload** (subida de archivos)
- **Slider** (deslizador)

## Botones

- **Icon** (ícono)
- **Avatar**
- **Chips** (etiquetas)
- **Button** (botón)
- **Radio button** (botón de opción)
- **Switch button** (interruptor)

## Navegación

- **Link** (enlace)
- **Menu** (menú)
- **Hamburger menu** (botón de hamburguesa)
- **Breadcrumb** (ruta de migas de pan)
- **Tabs** (pestañas)

## Progreso

- **Progress bar** (barra de progreso)
- **Stepper** (pasos)

## Notificaciones

- **Tooltip** (información sobre herramienta)
- **Snackbar** (notificación emergente)
- **Alert** (alerta)
- **Modal**

## Visuales

- **Table** (tabla)
- **Card** (tarjeta)
- **Badge** (insignia)
- **Graphic** (gráfico)
- **Accordion** (acordeón)

## Imágenes

- **Carrousel** (carrusel)
- **Map** (mapa)
